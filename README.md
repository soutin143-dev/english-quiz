# english-quiz
# 英単語クイズ（English Vocabulary Quiz）

このプロジェクトは、英単語レベル判定と学習用のクイズアプリです。  
HTML、CSS、JavaScript を使って作成しており、スマホやPCのブラウザで動作します。

---

## 🚀 プレイ方法

👉 [クイズをプレイする](https://あなたのユーザー名.github.io/english-quiz/)

1. ページにアクセスすると、まず説明画面が表示されます。  
2. 「クイズを始める」ボタンを押すと、英単語クイズが始まります。  
3. 全問解答後、結果が自動的に表示されます。

---

## 🧠 使用データ

- 本クイズの単語レベル分類には、**新JACET8000** の情報を参考にしています。  
- 新JACET8000は教育・研究目的で利用される語彙リストであり、  
  本アプリでは**データそのものを公開せず、学習済みモデルにのみ利用**しています。

> ⚠️ 著作権に関して：  
> 新JACET8000のデータを再配布することは行っていません。  
> このアプリは教育・研究目的で開発されています。
